To start: run main.py

Controls:
  - wasd: movement
  - space: jump
  - shift: dash
  - q/e: prev/next weapon
  - 1-8: weapons 1-8
  - o: fire
  - p: pause
  - v: weapon menu
  - esc: close program
  ____
  - 9: console
  - esc: close console
  - enter: send command in console

For hitboxes edit the flags.json file.
